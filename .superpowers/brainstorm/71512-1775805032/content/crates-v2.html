│   │

│   │   └── src/lib.rs
│   │
│   ├── yeek-adapter-codex/          # Codex CLI adapter
│   │   └── src/lib.rs
│   │

│   │   ├── src/
│   │   │   ├── lib.rs

Crate 依赖关系图


                          yeek-adapter-trait                       
                          ↑     ↑     ↑     ↑                       
        adapter-claude   adapter-codex     yeek-core             
                                              ↑                        
                                        yeek-cli    yeek-tui           

yeek-adapter-trait

只有 trait 定义和通用类型，零依赖。新增 adapter 只需依赖这个 crate。

adapter-claude / adapter-codex

各自独立。修改 Claude adapter 不会影响 Codex。可以独立发版和测试。

yeek-core

依赖 adapter-trait（编译期）+ 各 adapter（运行时注册）。Registry 动态组装所有 adapter。

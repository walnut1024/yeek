Crate 依赖关系

yeek-cli ──→ yeek-core
yeek-tui ──→ yeek-core

# yeek-core 的依赖：
rusqlite (SQLite) · zstd (压缩) · serde/serde_json · uuid · notify (watcher)
tokio (async runtime)

# yeek-cli 额外依赖：
clap (CLI 解析) · comfy-table (表格输出)

# yeek-tui 额外依赖：
ratatui (TUI) · crossterm (终端后端) · tui-tree-widget (项目树)

设计要点

  • yeek-core 独立于 UI — 可以作为库被其他项目引用
  • CLI 和 TUI 共享 core — 所有逻辑只写一次
  • Adapter 独立可测 — 每个 adapter 可以用 fixture JSONL 做单元测试
  • DB migration 独立管理 — SQL 文件方便版本控制和 review

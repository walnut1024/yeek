CLI & TUI 交互设计

两种交互模式共享同一套 Core API

CLI 子命令

yeek sync                     # 扫描所有 agent，增量导入新 session
yeek sync --agent claude_code     # 只同步指定 agent
yeek sync --watch             # 启动 file watcher 持续同步

yeek list                    # 列出所有 session（默认当前项目）
yeek list --all                # 所有项目的 session
yeek list --agent codex         # 只看某个 agent 的
yeek list --tag "debugging"      # 按标签过滤

yeek show <session-id>          # 显示 session 详情和对话内容
yeek resume <session-id>         # 用原始 agent 恢复 session
yeek resume <id> --agent codex    # 跨 agent 恢复（自动转换格式）

yeek share <session-id> --to codex   # 导出为指定 agent 格式
yeek delete <session-id>         # 从 yeek 删除（可选同时删原始文件）
yeek tag <session-id> "refactor"   # 给 session 打标签
yeek search "error handling"      # FTS 全文搜索

yeek tui                     # 启动交互式 TUI
